# 🤔 Resolvers

Graphql에서 데이터를 가져오는 구체적인 과정을 직접 구현해야 하는데 이는 Resolver가 담당한다.  

Schema가 데이터를 설명하는 담당 이라고 한다면 Resolver는 데이터를 가저오는 방법을 구현하는 담당이다.
```javascript
// 리솔버
const resolvers = {
  Query: {
    people: () => people, // 자자 people 가져오자!!
    person: (_: any, { id }: any) => getById(id), // 자자 people 아이디로 걸러 가져오자!
  }
}

```
#  

```
//스키마
type Person {
  id: String! // person 데이터 구조 설명
  name: String! // !는 repuire 
  age: Int!  
}

type Query {
  people: [Person]! // [person]은 people 배열
  person(id: Int!): Person // filter로 걸러 가져오기 
}
```

- 🚧 Resolver가 데이터를 어떻게 가져올지 구현하는 담당
