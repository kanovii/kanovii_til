# 🤔 Mutation

Graphql에서 데이터를 가져오는(fetching)부분 말고 대부분은 Mutation 타입의 필드가 담당한다.
  
Mutation은 기본적으로 Query와 동일하게 객체 타입이고 필드를 가지며 동일한 방식으로 작동한다. 


```javascript
// TS
const resolvers = {
  Query: {
    movies: () => movies, // fetch
    movie: (_: any, { id }: any) => getById(id), // filter
  },
  Mutation: {
    addMovie: (_: any, { title, score }: any) => addMovie(title, score), //추가
    deleteMovie: (_: any, { id }: any) => deleteMovie(id) //삭제
  }
}

```
#  

```
//스키마
type movies {
  id: Int! // person 데이터 구조 설명
  title: String! // !는 repuire 
  score: Int!  
}

type Query {
  movies: [Movie]! // [Movie]는 movies 배열
  Movie(id: Int!): Movie // filter로 걸러 가져오기 
}

type Mutation {
  addMovie(title: String!, score: Int!): Movie! //제목과 점수
  deleteMovie(id: Int!): Boolean! // 삭제 되면 true 출력
}

```

- 🚧 Mutation은 GET 이외의 요청을 대부분 수행
